# Estética Patt · Servicios Integrales

Sitio web del centro de estética **Estética Patt** (Villa Luro, CABA).

Tratamientos faciales, corporales, masajes y aparatología estética. Reserva de turnos por WhatsApp.

## Contenido

- `index.html` — la página completa. Es un único archivo autocontenido: incluye el HTML, los estilos (CSS), las animaciones (JS) y todas las imágenes embebidas. No necesita archivos externos.

## Cómo publicarlo

Cualquier hosting de sitios estáticos sirve. Solo hay que servir `index.html`:

- **Vercel / Netlify:** importar este repositorio y desplegar (detecta el `index.html` automáticamente).
- **GitHub Pages:** activar Pages sobre la rama principal.
